# 数据文件说明

本目录包含项目所需的数据文件。

## 必需文件

### 1. Hospital_Location.csv
医院位置数据，包含以下列：
- `name`: 医院名称
- `road_id`: 医院所在的SUMO路网edge ID
- 其他可选列：经纬度、地址等

示例格式：
```csv
name,road_id
协和医院,200042649
人民医院,200040849
第三医院,200063134
```

### 2. cases.txt
测试案例文件，每行包含一组事故点edge ID（空格分隔）

示例格式：
```
200001234 200005678 200009012 200003456 200007890
200002345 200006789 200000123 200004567 200008901
```

### 3. network.net.xml（需用户提供）
SUMO路网文件，需从您的SUMO项目中复制或生成。

获取方式：
- 使用SUMO的netconvert工具生成
- 从OpenStreetMap导出并转换
- 使用SUMO GUI创建

### 4. routes.rou.xml（需用户提供）
SUMO路由文件，包含基础交通流。

## 可选文件

- `experiment_results/`: 实验结果CSV文件
- 其他统计数据

## 注意事项

1. 路网文件和路由文件较大，不建议上传到Git仓库
2. 请确保医院road_id和事故点edge ID在路网中存在
3. 坐标系应与SUMO路网一致
